# Page Component

각 페이지만을 위한 컴포넌트로, 재사용 목적 보단 복잡한 UI에 대해 별도 독립된 관리 또는 묶기 위한 요소들입니다. (재사용 목적이여도 상관은 없음)

폴더명을 라우팅과 동일하게 맞추세요. 가령  /dashboard 페이지에 등장하는 컴포넌트의 폴더는 dashboard여야 합니다.

참고로 urlParameter는 고려하지 않습니다.
`ex) /search?q=...` 같은 건 전부 /search 폴더에 넣으면 됨.

여기에서 정의된 컴포넌트들은 해당 라우터 폴더 이름 이외에 다른 페이지에서 등장해선 안됩니다. 즉, header, sidebar와 같은 공통 레이아웃 요소들이나 검색창, 로딩바 같은 컴포넌트는 여러 페이지에서 사용된다면 `common`, `design`, `layout` 중 적절한 것을 선택해주세요
